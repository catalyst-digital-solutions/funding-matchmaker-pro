# Business Funding — Mesa Group Consulting

Standalone prototype for the **Business Funding** landing page (Mesa Group Capital).

Showcases 8 funding solutions from $1K to $100M for businesses at all stages, from startups to established companies, with access to 75+ lenders and $50M+ facilitated.

## Tech Stack
React 18 · TypeScript · Vite · Tailwind CSS · shadcn/ui
